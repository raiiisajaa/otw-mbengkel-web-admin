# OTW-MBENGKEL (Backend & Web Admin)

Proyek ini adalah sistem manajemen antrean booking servis mobil yang dikembangkan menggunakan framework Laravel.

### 🚩 Status Proyek: Work In Progress (WIP)

**Catatan:** Proyek ini masih dalam tahap pengembangan dan belum mencapai versi final 100%. Beberapa fitur backend dan integrasi API sedang dalam proses penyelesaian.

### 🛠️ Teknologi yang Digunakan

- **Framework:** Laravel 11
- **Bahasa:** PHP 8.3
- **Database:** MySQL
- **Otentikasi:** Laravel Sanctum (untuk API Mobile)
- **UI/UX:** Figma, Blade Template, Bootstrap 5

### 🚀 Fitur yang Sudah Diimplementasikan

- [x] Rancang Bangun Database (ERD).
- [x] Dashboard Monitoring Admin.
- [x] Manajemen Data Pelanggan & Kendaraan (One-Step Registration).
- [x] Integrasi Dasar REST API untuk Aplikasi Mobile.
- [x] Mekanisme Keamanan Token Base (Sanctum).

### 📝 Pengembangan Selanjutnya

- Penyelesaian integrasi penuh status booking secara real-time.
- Optimasi dokumentasi API menggunakan Postman/Swagger.
- Implementasi notifikasi otomatis.
